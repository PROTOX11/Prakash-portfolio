## SECTION: Projects

### Tickzen
Tickzen is an AI-Powered Task Management SaaS platform built to help teams work smarter, not harder.
- Built using React.js (frontend), Spring Boot (backend), PostgreSQL (database).
- Uses Gemini AI to auto-categorize, prioritize, and estimate task effort.
- Features real-time collaboration, role-based access control, and smart dashboards.
- Live demo: https://tickzen.in.net

### MedTrackFit
MedTrackFit is a smart AI-powered health and fitness tracking platform.
- Built using React.js, Node.js, MongoDB.
- AI features include personalized workout plans, diet analysis, and health insights.
- Users can track exercises, caloric intake, sleep patterns, and receive AI recommendations.
- Live demo: https://medtrackfit.in.net

### Vartalap
Vartalap is a real-time social chat application.
- Built using React.js, Node.js, Socket.IO, MongoDB.
- Supports real-time direct messaging, group chats, and media sharing.
- Features user authentication, profile management, and message notifications.
- Live demo: https://vartalap.in.net

### Aura Elysian
Aura Elysian is a premium e-commerce platform for fashion and lifestyle products.
- Built using React.js, Node.js, and PostgreSQL.
- Features product catalog, cart management, secure checkout, and admin dashboard.
- Live demo: https://auraelysian.shop

### Page Crafter
Page Crafter is a drag-and-drop website homepage builder.
